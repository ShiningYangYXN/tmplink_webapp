API Key
上传文件
POST https://tmp-cli.vx-cdn.com/app/upload_cli
Content-Type application/x-www-form-urlencoded

此接口即为 CLI 上传文件的接口，因此使用标准表单来提交需要上传的文件，可以使用 curl 或者 wget 等常用工具来完成上传。下面的代码是使用 curl 上传时的例子：

curl -k -F "file=@/root/test.bin" -F "key=API key" -X POST "https://tmp-cli.vx-cdn.com/app/upload_cli"
参数
  • key API Key
  • mrid 文件有效期
  • model 文件有效期
    • 99 永久
    • 0 这个文件在 24 后将被自动销毁
    • 1 有效期：3天
    • 2 有效期：7天
返回值

Status:

  • 1 : 上传完成，此时返回的 Data 数据为文件的 UKEY。
  • 0 : 上传失败，无效的请求。
  • 2 : 上传失败，文件大小超过限制。
  • 3 : 上传失败，服务器繁忙，无法处理此次请求。
  • 4 : 上传失败，私有空间不足，无法存储文件。
  • 5 : 上传失败，目前上传的文件总量已经超出了当日配额。存储至私有空间时不受限制。
  • 6 : 上传失败，API Key 已失效。

Data : String

  • 上传成功时，返回的数据为文件的 UKEY。上传失败时，返回数据是错误信息。
获取直链流量配额
POST https://tmp-api.vx-cdn.com/services/direct
Content-Type application/x-www-form-urlencoded
参数
  • action quota
  • key API Key
返回值

Status:

  • 1 : 有数据
  • 0 : 没有数据

    • Data : Array

  • quota : 已购买的配额
  • quota_free : 月度免费配额

    • 获取仓库文件列表
    POST https://tmp-api.vx-cdn.com/services/direct
    Content-Type application/x-www-form-urlencoded
    参数
    • action list_of_workspace
    • key API Key
    • page 页码，每页显示 1000 条数据
    返回值

    Status:

  • 1 : 有数据
  • 0 : 没有数据

    • Data : Array

  • ukey : 文件 UKEY
  • name : 文件名
  • size : 文件大小

    • 获取直链文件列表
    POST https://tmp-api.vx-cdn.com/services/direct
    Content-Type application/x-www-form-urlencoded
    参数
    • action list_of_direct
    • key API Key
    • page 页码，每页显示 1000 条数据
    返回值

    Status:

  • 1 : 有数据
  • 0 : 没有数据

    • Data : Array

  • dkey : 直链 DKEY
  • link : 下载链接
  • name : 文件名
  • size : 文件大小
  • etime : 链接有效期

    • 添加直链
    POST https://tmp-api.vx-cdn.com/services/direct
    Content-Type application/x-www-form-urlencoded
    参数
    • action link_add
    • key API Key
    • ukey 文件 UKEY
    • vaild_time （可选参数）链接有效期，单位为分钟，默认为 0 ，一直有效
    返回值

    Status:

  • 1 : 完成
  • 0 : 失败
  • 1001 : 失败，文件不存在
  • 1002 : 失败，内部错误

    • 移除直链
    POST https://tmp-api.vx-cdn.com/services/direct
    Content-Type application/x-www-form-urlencoded
    参数
    • action link_del
    • key API Key
    • dkey 直链 DKEY
    • delete 是否同时删除文件，如果设定为 true，将删除对应的文件，其它设定为此文件的直链也会一同删除。默认为 false。
    返回值

    Status:

  • 1 : 完成
  • 0 : 失败
  • 1003 : 失败，没有找到对应的直链
