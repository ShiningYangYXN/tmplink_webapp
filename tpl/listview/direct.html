欢迎使用 API
API Key
获取直链流量配额
POST https://tmp-api.vx-cdn.com/services/direct
参数 返回值
  • action quota
  • key API Key

Status:

  • 1 : 有数据
  • 0 : 没有数据

    • Data : Array

  • quota : 已购买的配额
  • quota_free : 月度免费配额

    获取仓库文件列表
    POST https://tmp-api.vx-cdn.com/services/direct
    参数 返回值
    • action list_of_workspace
    • key API Key
    • page 页码，每页显示 1000 条数据

    Status:

  • 1 : 有数据
  • 0 : 没有数据

    • Data : Array

  • ukey : 文件 UKEY
  • name : 文件名
  • size : 文件大小

    		•
    获取直链文件列表
    POST https://tmp-api.vx-cdn.com/services/direct
    参数 返回值
    • action list_of_direct
    • key API Key
    • page 页码，每页显示 1000 条数据

    Status:

  • 1 : 有数据
  • 0 : 没有数据

    • Data : Array

  • dkey : 直链 DKEY
  • link : 下载链接
  • name : 文件名
  • size : 文件大小

    		•
    添加直链
    POST https://tmp-api.vx-cdn.com/services/direct
    参数 返回值
    • action link_add
    • key API Key
    • ukey >文件 UKEY

    Status:

  • 1 : 完成
  • 0 : 失败
  • 1001 : 失败，文件不存在
  • 1002 : 失败，内部错误

    		•
    移除直链
    POST https://tmp-api.vx-cdn.com/services/direct
    参数 返回值
    • action link_del
    • key API Key
    • dkey 直链 DKEY
    • delete 是否同时删除文件，如果设定为 true，将删除对应的文件，其它设定为此文件的直链也会一同删除。默认为 false。

    Status:

  • 1 : 完成
  • 0 : 失败
  • 1003 : 失败，没有找到对应的直链
